# AITranscriptionSharp

AITranscriptionSharp는 OpenAI의 API를 사용하여 음성을 텍스트로 변환하는 WPF 애플리케이션입니다.

## 주요 기능

- OpenAI API 키 입력 및 저장
- AI 모델 선택 (whisper-1, gpt-4o-mini-transcribe, gpt-4o-transcribe)
- 단축키(Ctrl + Shift + A)를 사용하여 녹음 시작/중지
- 마이크 아이콘을 클릭하여 녹음 시작/중지

## 요구 사항

- .NET 9
- OpenAI API 키

## 설치 및 실행

1. 이 저장소를 클론합니다.

2. Visual Studio 2022에서 솔루션 파일(`.sln`)을 엽니다.

3. 프로젝트를 빌드하고 실행합니다.

## 사용 방법

1. 애플리케이션을 실행합니다.

2. OpenAI API 키를 입력하고 `Save API Key` 버튼을 클릭하여 저장합니다.

3. AI 모델을 선택합니다.

4. 단축키(Ctrl + Shift + A)를 사용하거나 마이크 아이콘을 클릭하여 녹음을 시작/중지합니다.
